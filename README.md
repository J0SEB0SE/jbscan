# jbscan
this is a lightweight network scanning tool built in Python. Designed with clarity and accessibility in mind, it provide clean, human-readable output for open ports, services, OS details and protocols. Ideal for cybersecurity students, it includes:- Simple CLI interface with educational output, support (Windows/Linux), and Scans single IP
